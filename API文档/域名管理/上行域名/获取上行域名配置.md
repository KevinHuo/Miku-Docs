# 获取上行域名配置
## 描述
获取指定直播空间中某个上行域名的详细配置信息

## 接口信息
- **请求方法**: GET
- **请求路径**: /
- **请求参数**:
  - **pushDomainConfig** (string) - 域名配置，必需
  - **name** (string) - 域名，必需  
    示例值: `www.example.com`


## 请求参数
### Header参数
| 参数名 | 位置 | 类型 | 必填 | 示例值 | 描述 |
|:---|:---|:---|:---|:---|:---|
| Host | Header | string | 是 | `<bucket>`.mls.cn-east-1.qiniumiku.com | 请求的服务域名 |
| Authorization | Header | string | 是 | `<QiniuToken>` | 管理凭证 QiniuToken |

## 访问域名
```go
<bucket>.mls.cn-east-1.qiniumiku.com
```

## 认证授权
```go
Authorization: <QiniuToken>
```
- **QiniuToken 详细描述请查阅**: [这里](/mikustream/api/12893/mikustream-live-http-requests-authentication)

## 请求示例
```http
GET /?pushDomainConfig&name=www.example.com HTTP/1.1
Host: test.mls.cn-east-1.qiniumiku.com
Authorization: <QiniuToken>
Content-Type: application/json

{}
```

## 返回响应

### 响应状态码
- **请查阅**: [这里](/mikustream/api/12894/mikustream-live-error-code)

### 响应示例
```json
{
  "auth": {
    "type": "",
    "enable": false,
    "primaryKey": "",
    "secondaryKey": "",
    "expireSeconds": 0
  },
  "bframeConfig": {},
  "certificateID": "",
  "cname": "www.example.com.publish.mikudns.com",
  "connectId": "NioAAF8KkEhOE3kY",
  "creationDate": "2025-11-18T18:32:15.357+08:00",
  "domain": "www.example.com",
  "enable": true,
  "lastModified": "2025-11-18T18:32:15.357+08:00",
  "type": "pushRtmp"
}
```
### 响应参数
| 参数名称 | 类型 | 说明 |
|:---|:---|:---|
| auth | object | 防盗链配置，[详见auth对象](#auth-object) |
| bframeConfig | object | B帧配置信息 |
| certificateID | string | HTTPS证书ID |
| cname | string | CNAME域名别名 |
| connectId | string | 请求连接的唯一标识符 |
| creationDate | string | 域名创建时间，遵循ISO 8601格式 |
| domain | string | 域名地址 |
| enable | boolean | 域名是否启用 |
| lastModified | string | 最后修改时间，遵循ISO 8601格式 |
| type | string | 域名类型（如pushRtmp、liveRtmp等） |

<a id="auth-object"></a>
### auth对象
| 参数名称 | 类型 | 说明 |
|:---|:---|:---|
| type | string | 防盗链类型 |
| enable | boolean | 是否启用防盗链功能 |
| primaryKey | string | 防盗链主密钥 |
| secondaryKey | string | 防盗链从密钥（备用） |
| expireSeconds | integer | 防盗链链接的有效期，单位为秒 |
