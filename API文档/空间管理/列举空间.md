# 列举空间
## 描述
获取当前账号下所有空间（Bucket）的列表信息。

## 接口信息
- **请求方法**: GET
- **请求路径**: /

## 请求参数
| 参数名 | 位置 | 类型 | 必填 | 示例值 | 描述 |
|:---|:---|:---|:---|:---|:---|
| Host | Header | string | 是 | mls.cn-east-1.qiniumiku.com | 请求的服务域名 |
| Authorization | Header | string | 是 | `<QiniuToken>` | 管理凭证 QiniuToken |

## 访问域名
```go
mls.cn-east-1.qiniumiku.com
```

## 认证授权
```go
Authorization: <QiniuToken>
```
- **QiniuToken 详细描述请查阅**: [这里](/mikustream/api/12893/mikustream-live-http-requests-authentication)

## 请求示例
```http
GET / HTTP/1.1
Host: mls.cn-east-1.qiniumiku.com
Authorization: <QiniuToken>
Content-Type: application/json

{}
```

## 返回响应

### 响应状态码
- **请查阅**: [这里](/mikustream/api/12894/mikustream-live-error-code)

### 响应示例

```json
{
  "buckets": [
    {
      "name": "test",
      "region": "cn-east-1",
      "creationDate": "2019-01-01T00:00:00.000+08:00",
      "status": "Enabled"
    }
  ],
  "connectId": "QXV0aG9yaXphdGlvbg"
}
```
### 响应参数
| 参数名称 | 类型 | 说明 |
|:---|:---|:---|
| buckets | []object | 空间列表，[详见bucket对象](#bucket对象) |
| connectId | string | 连接 id |

#### bucket对象
| 参数名称 | 类型 | 说明 |
|:---|:---|:---|
| name | string | 空间名称 |
| region | string | 区域 |
| status | string | 状态 （Enabled：可用；Disabled：不可用；Banned：禁用） |
| creationDate | string | 创建时间（ISO 8601） |
