# 创建空间
## 描述
创建一个新的空间（Bucket）。
创建后会自动分配区域和存储配置。

## 接口信息
- **请求方法**: PUT
- **请求路径**: /

## 请求参数

|   参数名	|   位置	 |   类型	   |必填	 |示例值	|描述|
|:--------- |:--------|:-----------|:-----|:---------|:---|
|   Host     |	Header|	string   |	是   | `<bucket>`.mls.cn-east-1.qiniumiku.com|	请求的服务域名，需包含要创建的空间名称|
|   Authorization | Header | string | 是 | `<QiniuToken>` |  管理凭证 QiniuToken |


## 访问域名
```go
<bucket>.mls.cn-east-1.qiniumiku.com
```

## 认证授权
```go
Authorization: <QiniuToken>
```
- **QiniuToken 详细描述请查阅**: [这里](/docs/public/请求鉴权)

## 请求示例
```http
PUT / HTTP/1.1
Host: test.mls.cn-east-1.qiniumiku.com
Authorization: <QiniuToken>
```

## 返回响应

### 响应状态码
- **请查阅**: [这里](/docs/public/错误返回)

### 响应示例

```json
{
  "name": "test",
  "region": "ap-1",
  "status": "Enabled",
  "creationDate": "2004-06-23",
  "connectId": "QXV0aG9yaXphdGlvbg"
}
```
### 响应参数
|   参数名称   |   类型   |    说明    | 是否必需 |
|:---------|:--------|:-----------|:----------|
| name     |  string |   空间名称  | 必需 |
| region   |  string |   区域  | 是 |
| status   |  string |   状态（Enabled：可用、Disabled：不可用、Banned：禁用） |  必需 |
| creationDate | string | 创建时间 |  必需 |
| connectId  |  string | 连接 id |  必需 |
