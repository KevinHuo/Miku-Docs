# 导播台

portal 导播台地址：  
https://portal.qiniu.com/mikustream/caster  

> 新建的导播台默认处于 **运行状态**，每次运行时长为 **1 小时**。  
> 导播台按运行时长计费，具体价格可参考 [价格详情](https://www.qiniu.com/prices/mikustream)。

# 设置导播台

![示意图](https://docs.qnsdk.com/miku_4f1c9b92e7fd4b9d8c3a12f94a6d58e1.png)

进入任一导播台的详情页面，可进行以下配置：

## 1. 临时令牌
用于生成一次性的加密登录链接。
- 令牌在设定的有效期内可直接访问导播台，无需输入密码。

## 2. 资源存储

- 创建空间：https://portal.qiniu.com/kodo/overview  
- Kodo 文档：https://developer.qiniu.com/kodo  
配置完成后可在导播台中直接上传本地静态资源（图片、视频等）。

## 3. 输入源设置
- 支持 **8 路输入信号源**。  
- 输入可以是：
  - 已存在的直播流  
  - 点播视频（VOD）  
- 可在导播台界面快速切换信号源。

## 4. PGM 输出设置
- 配置导播混流输出的目标推流地址（RTMP）。

> 注：当前导播台对输入与输出 URL 均有限制。

---

# 进入导播台

可通过两种方式进入：

## 1. 使用静态鉴权 Key
- 作为密码，直接登录导播台。

## 2. 使用临时令牌链接
- 通过临时令牌生成的访问地址，免密进入。  
- 链接有时间限制，到期自动失效。
